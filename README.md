# 🎧 SOLPLY-ANDROID
### 나만의 시간을 즐겁게,<br> 솔플리
<br>

## 🥅 서비스 목표
### 혼자서도 편안하게 머무를 수 있는 장소를 쉽게 찾고, 자신의 취향대로 혼놀 코스를 설계할 수 있도록 도와줍니다.
<br>

## ✒️ 주요 기능 소개
### 1. 맞춤 큐레이션 탐색
#### ✅ 장소 / 방문 목적 / 기타 특징에 기반한 필터링 기능
#### ✅ 내 취향 키워드가 반영된 추천 컨텐츠 노출
<br>

### 2. 장소 뷰 & 코스 뷰 선택
#### ✅ 장소 단건 or 코스 전체 저장 기능
<br>

### 3. 수집함 기능
#### ✅ 마음에 드는 장소와 코스 저장 및 활용
#### ✅ 저장한 코스를 커스텀하며 나만의 코스로 재탄생
<br>
